About

Portal Stories: Mel Speedrun Mod is a mod tailored specifically for speedrunners. Portal Stories: Mel Speedrun Mod transforms the original campaign by streamlining levels, elimination cutscenes, and removing other annoyances, making the experience smoother and more engaging. Tried and true to the original Portal 2 Speedrun Mod, the mod was made with VScript, and the source code can be found on our GitHub.

Installation

  • For now, this mod is for Windows and Linux only. Sorry, Mac users :/. Make sure you have Portal Stories: Mel downloaded on your device (Portal 2 being installed is NOT a requirement).
  • Unzip 'Portal Stories Mel Speedrun Mod' folder into the 'sourcemods' directory (it's default location is 'Steam/steamapps/sourcemods/').
  • Restart Steam.
  • A new game called "Portal Stories: Mel Speedrun Mod" should appear in your library.

Keep in mind that this mod is built on janky code to remove/speed up elements in a map, so if you encounter an issue, try one of these things:

  • Launching the main game, then the mod
  • Not using any other extensions (like SAR, for instance) because they might not like each other
  • Updating other extensions (like SAR), because they might be the one causing trouble
  • Taking a hot bath and at least 30 minutes of walk in fresh air#stayathome
  • committing a jump from a ground floor window

If none of these work, seek help in the P2SR Discord Server

Have fun speedrunning!
