What is Portal Stories: Mel Speedrun Mod?

Welcome to the official site for the Portal Stories: Mel Speedrun Mod! The mod is meant to change each individual level in Portal Stories: Mel to make them faster, using only vscript! This means that no recompiling of maps was done, which can cause many errors and inconsistencies. It also greatly reduces file sizes. Things such as skipping cutscenes and making doors open faster are things included in this mod. The aim of this mod is to make the Mel speedrun more enjoyable, with many quality of life improvements and skipped cutscenes/downtime.
